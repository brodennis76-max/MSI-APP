# QR Code Images

Upload your QR code PNG files here.

## How to use:

1. **Add your PNG files** to this folder
2. **Commit and push** to GitHub
3. **Get the raw URL**:
   - Go to your file on GitHub
   - Click the "Raw" button
   - Copy the URL (format: `https://raw.githubusercontent.com/brodennis76-max/MSI-APP/main/msi-expo/qr-codes/your-file.png`)
4. **Paste the URL** in the "Scanner QR Code Image URL" field when editing a client

## Example URL format:
```
https://raw.githubusercontent.com/brodennis76-max/MSI-APP/main/msi-expo/qr-codes/scanner-config-1.png
```

## Notes:
- Files must be PNG format
- Keep file names descriptive (e.g., `scanner-config-store1.png`)
- The PDF generator will use the image URL if provided, otherwise it will generate from the text QR code data
